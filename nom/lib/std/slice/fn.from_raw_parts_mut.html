pub unsafe fn from_raw_parts_mut<'a, T>(p: *mut T, len: usize) -> &'a mut [T]

Performs the same functionality as from_raw_parts, except that a mutable slice is returned.

This function is unsafe for the same reasons as from_raw_parts, as well as not being able to provide a non-aliasing guarantee of the returned mutable slice. p must be non-null even for zero-length slices as with from_raw_parts.
