Module gutenberg_post_parser::parser[−][src]

The Gutenberg post parser.

The Gutenberg post parser is a parser combinator. Thus it provides mulitple parsers, aka combinators. They are based on the nom project.

The writing of combiantors heavily relies on Rust macros. Don't be surprise! To learn more, consult the documentation.
